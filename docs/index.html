This page helps you quickly install the widget tool script which will allow you to instantly install and update the Fordpass Widget with a single tap.
No more copying and pasting!

GitHub Project Readme

Installation

Install the widget tool in just three steps.

NOTE: iOS, iPadOS, and macOS devices only!

1. Install Scriptable

Download Scriptable ↗

If you have not downloaded Scriptable from the AppStore yet, please click on the download button.

2. Copy Installer Script

Copy Installer

Copy the widget tool installer script to your device clipboard by clicking this button.

3. Install Ford Widget Tool

Open Scriptable

Opens a new Scriptable script, paste the code and press run (▶).
After that you will see the script close and then the FordWidgetTool will appear and run the install of the FordPass Widget.
